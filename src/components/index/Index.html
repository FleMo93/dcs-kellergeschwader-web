WIR ÜBER UNS

Die Kellerkompanie, eine im Aufbau befindliche DCS Community, fliegt seit 2019 in gemütlicher Runde zusammen. Bei uns ist jeder, vom Anfänger bis zum echten Piloten willkommen. Es herrschen keine Verpflichtungen und Verbindlichkeiten zur Anwesenheit oder ähnlichem, wie es bei anderen Communities der Fall ist.

WO FINDET IHR UNS

Die meiste Kommunikation findet auf unserem Discord Server statt. Einfach beitreten und Spaß haben.

SERVER

Wir betreiben einen dedizierten öffentlichen Server mit abwechlungsreichen Missionen, der konstant weiterentwickelt wird. Er verfügt über ein Votingsystem, SRS und zufälligem Wetter beim Missionsstart.
